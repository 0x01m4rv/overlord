# create-domain

Adds a domain list to Digital Ocean

# Arguments

| Name                      | Value Type   | Description
|---------------------------| ------------ | -----------
|`name`                     | List(string) | The domain names to add 
|`counter`                  | Integer      | The domain names to add 

# Output
| Name                      | Value Type | Description
|---------------------------| ---------- | -----------
|`domain_name`              | LIST       | Domain name list
